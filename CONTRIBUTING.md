# Contributing to Pharmaceutical QMS

## كيفية المساهمة | How to Contribute

### الإبلاغ عن الأخطاء | Reporting Bugs
- استخدم قسم [Issues](../../issues)
- استخدم قالب "Bug Report"
- أرفق خطوات إعادة الإنتاج

### اقتراح ميزات | Suggesting Features
- افتح [Issue](../../issues/new) مع وسم `enhancement`
- اشرح الفائدة المتوقعة

### Pull Requests
1. Fork المستودع
2. أنشئ فرعاً: `git checkout -b feature/AmazingFeature`
3. التزم بـ [Conventional Commits](https://conventionalcommits.org)
4. اختبر تغييراتك
5. افتح Pull Request

### معايير الكود | Code Standards
- **Python**: PEP 8, Black formatter
- **JavaScript**: ESLint, Prettier
- **Git**: Conventional Commits

### الأمان | Security
- لا ترفع بيانات سرية
- استخدم `.env` للمتغيرات
- راجع `SECURITY.md`
