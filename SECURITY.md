# Security Policy

## Reporting Security Vulnerabilities

إذا اكتشفت ثغرة أمنية، يرجى:
1. عدم فتح Issue عام
2. إرسال بريد للمشرف مباشرة
3. الانتظار 90 يوماً قبل النشر العلني

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ Yes    |
| < 1.0   | ❌ No     |

## Security Measures
- تشفير البيانات الحساسة
- التحقق من الصلاحيات
- حماية CSRF مفعلة
- Rate Limiting
